查看帮助 `F1`
新标签页 `Ctrl+T`
编辑文档属性 `Ctrl+;`
查看当前文件大纲 `Ctrl+Q`
切换编辑/阅读视图 `Ctrl+E`
插入链接 `Ctrl+K`
打开设置 `Ctrl+，`
返回上个页面 `Ctrl+Alt+←`
导出为PDF `Ctrl+Alt+P`
导出为HTML(Pando目录) `Ctrl+Alt+↑`
嵌入内容 `Ctrl+Alt+N`
